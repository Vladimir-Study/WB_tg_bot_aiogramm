Добавить API токен

  1. Перейдите в личный кабинет Wildberries (данное действие необходимо выполнять владельцу кабинета)
    2. Профиль -> Настройки -> Доступ к API
  2. Введите имя токена, например Astra
  3. Выберите “Контент” и “Вопросы и отзывы”
  4. Нажмите “создать токен”
